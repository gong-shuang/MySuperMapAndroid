# intelligencetour

## 范例简介

在AR实景中查找地点，显示地点详细信息，并展示导航路线，功能有POI搜索，AR空间展示，路径分析

## 示例数据

数据目录：在线google地图

## 关键类型/成员

ArView

ARRendererInfoUtil	

GeoObject
	
OnClickArObjectListener

GeocodingData

WorldLocationChanged
	
## 使用步骤

1. 上拉，选择分类
2. 在摄像头中展示查询结果
3. 点击POI投射，展示详细信息
