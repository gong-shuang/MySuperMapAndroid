# onlinelicense

## 范例简介
示范公有云许可的使用，包括查询，激活，退出等功能。


## 关键类型
CloudLicenseManager

## 使用步骤

1. 点击【查询模块】按钮，查询许可信息。
2. 点击【许可激活】按钮，激活许可。
3. 点击【打开地图】按钮，展示地图。
4. 点击【退出登录】按钮，退出登录，同时许可无效。

##注意事项

1. 必须开启WIFI
2. 如果许可网络断开连接，许可将进行回收，无法使用移动端功能
3. 如果为正式许可，需要勾选"核心开发模块"、"核心运行模块"进行开发
